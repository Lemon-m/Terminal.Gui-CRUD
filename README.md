# Terminal.Gui CRUD C# App

## Requirements

- .NET SDK 8.0 or later

## How To Run

### 1. Clone the repository
```Bash
git clone https://github.com/Lemon-m/Terminal.Gui-CRUD.git
cd Terminal.Gui-CRUD
```

### 2. Restore dependencies
```Bash
dotnet restore
```

### 3. Run the app
```Bash
dotnet run
```
